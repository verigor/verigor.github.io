MY UNREAL ENGINE UI DEMO

I've implemented fully functional skill tree widget and skill system.
Visually it references Assassin Creed Origins tree skill.
This project also contains save/load system

Skill Tree Widget

Widget shows a tree of connected skill slots. Each slot can be bought and upgraded. Hovering a slot causes popping up a hint widget.

Buying a skill

Buying a new skill unlocks new available skills and updates their connections. If a player doesn't have enough points he would be worn with a hint.

Upgrading a skill

You can upgrade skills if you have enough points. Pop-up hints helps player to see slot's level.

Game Save / Load

Skills are stored in SkillController actor component. It's state is restored when loading a game. Player can save a list of saves from the main menu.

Geting skill points

Skills and skill points are stored inside SkillController actor component (attached to PlayerController). Points can be obtained by collecting glowing cubes.
